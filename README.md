# rules-us-al

Alabama statutes and regulations in Akoma Ntoso XML format.

## Structure

```
rules-us-al/
├── statutes/       # Alabama Code sections in Akoma Ntoso XML
└── regulations/    # Alabama Administrative Code in Akoma Ntoso XML
```

## Format

Files follow the [Akoma Ntoso](http://docs.oasis-open.org/legaldocml/akn-core/v1.0/akn-core-v1.0-part1-vocabulary.html) standard for legal documents.

## Source

- Statutes: [Alabama Legislature](https://alison.legislature.state.al.us/code-of-alabama)
- Regulations: [Alabama Administrative Code](https://admincode.legislature.state.al.us/)
